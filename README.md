## Usage
- Open `index.html` in any modern browser; no build steps or dependencies.
- Edit copy or sections directly in `index.html` (hero, villas, services, amenities, process, testimonials, contact).

## Customization
- Replace the Unsplash background images in the villa cards by updating the `background-image` URLs inside the `.villa-visual` elements.
- Adjust the color system in the `:root` CSS variables to shift the visual direction.
- Update contact details in the `#contact` section and header CTA links to match your preferred inbox/phone.
